Welcome to PyISI's Documentation
==============================

PyISI is a modern Python library implementing the intrinsic signal imaging (ISI) analysis protocol. Developed in the Kim Laboratory at UC Santa Cruz, this project modernizes the original MATLAB codebase with contemporary software engineering practices, improved performance, and enhanced usability.
